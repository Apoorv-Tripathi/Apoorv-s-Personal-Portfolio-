Modern Portfolio Website – React + Bootstrap

A fully responsive and modern personal portfolio website built using React, Bootstrap, and custom gradients with subtle professional styling.
This website includes dedicated sections for Home, About, Skills, Projects, Education, and Contact, along with animated interactions and smooth UI elements.

⸻

🚀 Features

✅ Home Section
	•	Photo displayed on the right side with a unique transitional border
	•	Automatic typing text animation showing skills
	•	Subtle, professional gradient background

✅ About Section
	•	Detailed introduction
	•	Extended description of background, interests, and expertise
	•	Skills subsection designed with animated skill indicators

✅ Projects Section
	•	Interactive project cards with hover animations
	•	Each card includes project title, description, tech stack, and link

✅ Education Section
	•	Clean timeline format displaying academic journey

✅ Contact Section
	•	Two-column layout: photo + contact form
	•	Fully styled form with validation
	•	Improved contrast for perfect readability

✅ Additional UI Features
	•	Floating sticky navbar with margin for modern look
	•	Subtle professional gradients (yellow as base + supporting tones)
	•	Smooth scroll, hover effects, and animations
	•	Fully responsive across devices
🛠 Technologies Used
	•	React.js
	•	Bootstrap
	•	Custom CSS
	•	JavaScript
	•	React Typing Animation
	•	Responsive Web Design Principles
